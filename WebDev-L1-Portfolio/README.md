# Personal Portfolio Website

## Overview

This project is a responsive personal portfolio website developed using HTML5 and CSS3. It showcases my professional profile, technical skills, projects, and contact information. The portfolio is designed with a clean, modern layout and serves as my digital resume.

## Features

- Responsive design for desktop and mobile devices
- Professional hero section
- About Me section
- Technical Skills section
- Experience section
- Projects showcase
- Contact section
- Clean and modern user interface
- Smooth navigation between sections

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Poppins)

## Project Structure

```
WebDev-L1-Portfolio/
│── index.html
│── style.css
└── README.md
```

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.

Alternatively, use the Live Server extension in Visual Studio Code.
