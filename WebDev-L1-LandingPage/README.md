#  AI Learning Hub - Landing Page

##  Overview
AI Learning Hub is a responsive landing page designed using HTML5 and CSS3. It presents an attractive and modern interface for an AI learning platform with a hero section, feature highlights, about section, and footer.

##  Features
- Responsive design for desktop and mobile
- Sticky navigation bar
- Hero section with Call-to-Action button
- Features section
- About section
- Modern typography and color palette
- Clean and organized layout

##  Technologies Used
- HTML5
- CSS3

##  Project Structure
```
LandingPage/
│── index.html
│── style.css
└── README.md
```

##  How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html** or use **Live Server** in VS Code.
