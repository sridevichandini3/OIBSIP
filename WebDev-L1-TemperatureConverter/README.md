# Temperature Converter

## Overview

Temperature Converter is a responsive web application developed using HTML5, CSS3, and JavaScript. It allows users to convert temperature values between Celsius, Fahrenheit, and Kelvin with accurate calculations, input validation, and a clean user interface.

## Features

- Convert temperatures between Celsius, Fahrenheit, and Kelvin
- User-friendly interface
- Input validation for numeric values
- Absolute zero validation
- Responsive design for desktop and mobile devices
- Modern and clean layout

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.

Alternatively, use the Live Server extension in Visual Studio Code.
